Terminal

Best for: Deep work, coding with Claude Code, complex multi-step tasks.

What You Get

  • Cal's Memory — your personal context, preferences, and history
  • Cal's Personality — the same thinking partner behavior everywhere
  • Cross-Channel Context — Cal knows what you discussed earlier
  • Custom Tools — calendar, mail, notes, files, search, and any MCP servers you add
  • Claude Code's Power — file editing, git, code analysis, and large-context work

Web UI (PWA)

Best for: Quick access from any browser, install as a mobile app.

Features

  • Works in any browser
  • Installable as a Progressive Web App on your phone
  • Same session as all other channels
  • Clean interface for quick prompts and session review

Setup

No setup required. The Web UI runs when Cal Gateway starts.

# Install as PWA on iPhone
# 1. Open the Web UI URL in Safari
# 2. Tap Share → Add to Home Screen
# 3. Name it "Cal"

iMessage

Best for: On-the-go queries from Apple devices.

Features

  • Native Apple Messages integration
  • Works on Mac, iPhone, iPad, and Apple Watch
  • Messages are read locally from your Mac
  • Identity separation is configurable

Setup

  1. Install imsg: brew install imsg
  2. Grant Full Disk Access to imsg and node
  3. Configure identity in config/imessage.json
  4. Restart Gateway

Telegram

Best for: Remote access from any device with a configured bot.

Features

  • Mobile-friendly access when you are away from your Mac
  • Shared Cal session and memory
  • Works well for quick status checks, reminders, and brief research prompts

Setup

Create a Telegram bot, add the token to your local config, and restart the gateway.
