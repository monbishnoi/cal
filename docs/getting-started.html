Getting Started

Get Cal running locally.

Prerequisites

  • macOS (Sonoma or later recommended)
  • Node.js 18+ (brew install node)
  • Anthropic API key or a compatible Claude API endpoint
  • Claude Code for terminal/developer workflows
  • Prevent Sleep App if you want Cal to stay responsive while your Mac is unattended

Quick Start

1. Clone the Repository

git clone https://github.com/monbishnoi/cal.git
cd cal

2. Install Dependencies

npm install

3. Configure Environment

cp config/env.example config/.env

Edit config/.env:

# Required for Claude access
ANTHROPIC_API_KEY=your-key-here

# Optional: custom Claude-compatible endpoint
# ANTHROPIC_BASE_URL=https://your-provider.example/anthropic

# Optional: iMessage phone number
# IMESSAGE_ALLOWED_PHONE=+1XXXXXXXXXX

4. Install System Dependencies

# Calendar integration
brew install ical-buddy

# Notes integration
brew tap antoniorodr/memo && brew install antoniorodr/memo/memo

# iMessage integration (optional)
brew install imsg

5. Set Up Apple Shortcuts

  1. Open the shortcuts/ folder in Finder
  2. Double-click each .shortcut file to import
  3. First run will prompt for Calendar or Reminders permission; select "Always Allow"

6. Google Workspace MCP (Optional)

Enable this when you want Cal to work with Google Docs. Phase 1 supports Docs search, read, create, and confirmed non-destructive batch updates.

npm install -g @googleworkspace/cli
gws auth login

Then enable the google-workspace MCP server in your local jobs config. Writes require a confirmation token, and delete/share/permission/ownership tools are not exposed.

View the Google Workspace capability details →

7. Start Cal Gateway

Foreground (for testing):

node src/gateway.js

With pm2:

pm2 start src/gateway.js --name cal-gateway
pm2 save

8. Test It

cal
> What's on my calendar today?

Troubleshooting

Cal stops responding

Your Mac may have gone to sleep or the gateway may have stopped.

pmset -g log | grep -i sleep | tail -5
pm2 status cal-gateway

Solution: Keep the Mac awake for unattended access and restart the gateway if needed.

Gateway will not start 
tail -50 logs/gateway-error.log

# Common issues:
# - Missing ANTHROPIC_API_KEY
# - CAL_HTTP_PORT already in use
# - Missing node_modules
Calendar or Notes not working 
/opt/homebrew/bin/icalBuddy eventsToday
/opt/homebrew/bin/memo notes --no-cache

Check macOS privacy permissions for Terminal, Node, Shortcuts, Calendar, and Notes.
